# Elden Ring Route Tracker

Локальный checklist-трекер предметов Elden Ring на основе твоей Notion-структуры.

## Как открыть

Самый быстрый вариант: дважды кликни файл:

```text
OPEN_TRACKER.url
```

Он открывает основной трекер.

Открой файл:

```text
C:\Users\Erik\Documents\New project\index.html
```

Его можно просто дважды кликнуть в проводнике Windows. Приложение откроется в браузере.

## Что уже работает

- Поиск предметов, локаций и NPC.
- Фильтры по локации, категории и типу получения.
- Отметки `собрано / не собрано`.
- Сохранение прогресса в браузере.
- Редактирование предметов прямо в приложении.
- Добавление новых предметов вручную.
- Отдельный режим `Пополнить из Fandom`.
- Каталожный слой `Каталог Fandom` для предметов, которых пока нет в Notion.
- Панель `Покрытие базы`, которая показывает регионы без записей.
- Список локаций сгруппирован по большим регионам: точные места вроде `Замок Грозовой Завесы` остаются в карточках предметов, но не засоряют левый фильтр.
- Панель `Очередь Fandom` для быстрого перехода к неуточненным категориям.
- Режимы базы: `Вся база`, `Маршруты`, `Каталог Fandom`, `Нужно уточнить`.
- Кнопки режимов показывают текущие счетчики, включая очередь `Нужно уточнить`.
- Экспорт и импорт backup-файла с прогрессом и ручными правками.
- Ссылки на Fandom, карту или видео.

## Как редактировать предмет

1. Выбери предмет в центральном списке.
2. В правой панели нажми `Редактировать`.
3. Измени название, локацию, тип получения, описание или предупреждение.
4. Нажми `Сохранить изменения`.

Твои правки сохраняются локально в браузере. Оригинальные данные в самом файле не удаляются.

## Как добавить новый предмет

1. Нажми `Добавить предмет` в левой панели.
2. Заполни название, категорию, локацию, тип получения и короткое описание.
3. В поле `Ссылки` добавь Fandom, карту или видео.
4. Нажми `Добавить предмет`.

Новые предметы тоже сохраняются в браузере и попадают в backup.

## Как добавить ссылку на Elden Ring Fandom

В поле `Ссылки` используй формат:

```text
Fandom | https://eldenring.fandom.com/ru/wiki/Редувия
Map | https://mapgenie.io/elden-ring/maps/the-lands-between?locationIds=123
Video | https://youtu.be/example
```

Каждую ссылку нужно писать с новой строки.

## Backup

Кнопка `Экспорт прогресса` создает JSON-файл с:

- отметками собранных предметов;
- ручными правками описаний;
- измененными ссылками;
- предметами, которые ты добавил вручную.

Кнопка `Импорт backup` возвращает эти данные обратно в приложение.

## Как работать с Elden Ring Fandom

Да, данные можно брать с Elden Ring Fandom, но лучше переносить их в приложение как короткий пересказ:

- название предмета;
- категория;
- где найти или как получить;
- важные условия, которые можно пропустить;
- ссылка на Fandom-страницу как источник.

Не стоит переносить в приложение большие куски текста слово в слово. Лучше коротко переписать своими словами и оставить ссылку `Fandom | ...` в поле ссылок.

Если предмет фармится с врагов, которые встречаются во многих местах, в карточке лучше указывать одну рекомендуемую точку фарма: ближайшую к месту благодати, самую удобную для повторного захода или самую раннюю по маршруту. Остальные возможные места можно кратко упомянуть в заметке.

Серийные ресурсы прокачки и обычные крафтовые материалы не стоит держать в основном чеклисте. Категория `Материалы` полностью убрана из базы, чтобы не забивать маршруты кузнечными камнями, ландышами, травами, мхом, грибами, лапками, костями и похожими повторяющимися ресурсами.

Пример:

```text
Fandom | https://eldenring.fandom.com/ru/wiki/Редувия
```

## Процесс `Пополнить из Fandom`

1. Нажми `Пополнить из Fandom` в левой панели.
2. Вставь ссылку на страницу Elden Ring Fandom.
3. Скопируй короткий фрагмент со страницы или напиши свои заметки.
4. Нажми `Автозаполнить из Fandom-полей`.
5. Проверь название, категорию, локацию и описание.
6. Нажми `Добавить предмет`.

Этот процесс не копирует всю страницу автоматически. Он помогает быстро создать чистую запись: короткий пересказ плюс ссылка на источник.

## Если в Notion нет всех локаций

Это нормально. Приложение теперь не считает Notion полной базой игры.

В левой панели есть блок `Покрытие базы`. Он показывает:

- сколько ожидаемых регионов уже имеют записи;
- какие регионы пока пустые;
- какие регионы нужно проверить через Elden Ring Fandom.

Если ты выбираешь пустую локацию, приложение предложит `Пополнить из Fandom` или `Добавить предмет`.

## Что такое `Каталог Fandom`

`Каталог Fandom` - это черновой слой базы для категорий, которых пока нет в Notion или которые еще не перенесены полностью.

В него добавлены крупные списки:

- талисманы;
- духовный прах;
- заклинания;
- молитвы;
- комплекты брони;
- ключевые предметы.
- книги и справочники.

Такие записи уже можно искать, фильтровать и отмечать как собранные. Но их маршрут получения пока считается неуточненным. Для каждой записи есть ссылка на Elden Ring Fandom, а точное место получения можно добавить позже через `Редактировать`.

Когда предмет проверен по русскому Fandom, он переносится из `Каталог Fandom` в реальную локацию. Например, `Склад великого кувшина` теперь находится в `Звездные пустоши`, а не в общей очереди.

## Режимы базы

В левой панели есть блок `Режим базы`:

- `Вся база` показывает все записи.
- `Маршруты` показывает записи с уже заданной локацией.
- `Каталог Fandom` показывает черновые записи из Fandom-слоя.
- `Нужно уточнить` показывает важные маршрутные пробелы; `Инструменты`, золотые руны, руны героев и сетевые предметы полностью убраны из чеклиста.

Для работы над базой удобнее идти так: выбрать `Каталог Fandom`, открыть предмет, перейти по ссылке Fandom, уточнить место получения и нажать `Редактировать`.

## Очередь Fandom

В левой панели есть блок `Очередь Fandom`. Он показывает категории каталога и количество записей в каждой из них.

Нажми на категорию, например `Талисманы` или `Броня`, и приложение сразу переключится в режим `Каталог Fandom` с нужным фильтром. В правой панели у каталожного предмета появится чеклист: ссылка Fandom, реальный регион, конкретное место, тип получения и короткое описание.
